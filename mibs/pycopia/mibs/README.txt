All pre-compiled MIB files are installed seperately, as a tarball. The latest
versions are available here:

http://pycopia.googlecode.com/files/pycopia_mibs.tar.gz

or

http://www.pycopia.org/download/pycopia_mibs.tar.gz

or 

ftp://ftp.pycopia.org/pub/python/pycopia_mibs.tar.gz

After downloading that file do the following:

mkdir -f /var/tmp/mibs
cd /var/tmp
tar xzf /path/to/pycopia_mibs.tar.gz
