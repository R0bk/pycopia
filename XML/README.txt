===========
Pycopia-XML
===========

Work with XML in a Pythonic way.

Provides Python(ic) Object Model, or POM, for creating, inspecting, or
modifying basic XML documents. Partially validates documents. This framework
requires a DTD for the XML. No need to use string operations to construct XML
documents. The XML is partially validated against a pre-compiled DTD file.

There is a framwork for generating XHTML from Python code.
