===========
Pycopia-WWW
===========

Pycopia package for serving and fetching web pages.

Pycopia WWW tools and web application framework.  Provides FCGI servers,
XHTML page generator with functional style interfaces, and lightweight web
application framework. Designed to work closely with the lighttpd_ front-end
server.

There is a JSON proxy that enables the transparent transfer of data to and
from Python and Javascript.


.. _lighttpd: http://www.lighttpd.net/
